# Books Management

## Technologies
This is a web application built with:
* Java 20
* Spring Boot 3.1.0
* Spring MVC
* Spring Security
* Spring Data JPA
* Hibernate
* PostgreSQL
* Thymeleaf

## Functionalities of BookSite App
* Login
* Registration
* Adding a book
* Viewing books
* Adding books with Google Books API
